Git Bash - Terminal estendido para utilizar o uso do Git.

Comandos Úteis:

git clone - Clonar o link de repositório do GitHub para a máquina.
cd - Entrar em uma pasta.
ctrl + l - Limpar a tela (melhor para visualizar o Git).
git status - Verificar o Status.
git add . - Adicionar os arquivos alterados.
git commit -m "Anotações sobre as alterações" - Criar Commit. 
git push origin main - Finalizar sincronismo com o GitHub.
